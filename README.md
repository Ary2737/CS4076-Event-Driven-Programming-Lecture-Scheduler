# CS4076-Event-Driven-Programming-Lecture-Scheduler
This is a repo for one of my modules CS4076 (Event-Driven Programming) where I develop a TCP , Client-Server Lecture Scheduler application. In this project I have two applications which are....
 -- 24436143_Client: This is my Client GUI application where I use JavaFX and SceneBuilder to create the GUI for the client. Herre the client can preform many fesatures such as Adding a lecture , removing a lecture or displaying a lecture.
 -- 24436143_Server: This is my console server application where a TCP connection between the client and server applications is created and the client's requests are handled.


This project was devloped using the MVC (Model-View-Controller) model for both applications.

Technologies/Skills used in this project...

-- JavaFX 
-- Java
-- SceneBuilder (To create Client GUI using .fxml files)
-- Exception Handling
-- Network programming in TCP (Using Socket/ServerSocket classes)
-- Reading input/output.
